# Vehicle Parking Management System

A simple **Vehicle Parking Management System** developed using the **C Programming Language**.  
The program allows users to park vehicles, display parked vehicles, and calculate parking charges.

---

## Features

- Park vehicles
- Store vehicle records using file handling
- Display all parked vehicles
- Calculate parking charges
- Menu-driven interface

---

## Technologies Used

- C Programming
- File Handling
- Structures
- Menu Driven Program
